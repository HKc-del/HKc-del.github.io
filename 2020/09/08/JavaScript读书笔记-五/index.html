RegExp类型补充与其他

1.RegExp类型

定义式

pattern部分为任意的正则表达式，flags为标志（g全局模式,i不区分大小写,m多行匹配）

实例属性

global 是否设置了g flag

ignoreCase 是否设置了i flag

lastIndex 表示开始搜索下一个匹配字符位置

multiline 表示是否设置了mflag

source 正则表达式字符串

实例方法

exec()方法接受一个参数，即要被匹配的字符串。

返回的对象：

index 匹配的下标、lastIndex为下一次匹配开始的下标。

本身为一数组第一项包含着整个匹配的字符串，往后的下表存放着分别捕获的分组。

test()方法接受一个被测试的字符串，返回一个布尔值，判断是否存在匹配。

构造函数属性

input 最近一次要匹配的字符串

lastMatch最近一次的匹配项

lastParen最后一次匹配的捕获组

leftContext lastMatch之前的文本

multiline 布尔值，表示是否使用了多行模式

rightContext lastMatch之后的字符串

2.Function类型

函数本质上为一对象。JavaScript中无法进行函数重载。JavaScript解释器会优先解析函数声明，使其在语句中变得可用，而函数表达式的解释推迟到函数执行时进行。（函数声明提升）但是变量的声明过程并不会提升，所以下面的代码段会执行失败。

函数作为对象也可作为函数的返回值。

arguments和this属性

arguments包含着传入参数构成的数组，存在一个callee属性为一指针，指向这个函数。

this对象（环境对象）

caller属性调用该函数的对象名

apply与call方法

apply()接受两个函数参数，第一个为函数要调用的环境对象，第二个为参数数组(可以用arguments对象)，可以使用特定的环境对象调用函数。

严格模式之下，未指定环境变量对函数的调用不会被指定默认环境对象window，而为undefined。
