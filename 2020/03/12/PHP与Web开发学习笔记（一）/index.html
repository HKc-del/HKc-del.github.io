isset(var) Returns true if it exists and false otherwise.

unset(var) This function gets rid of the variable it is passed

bool empty(var) The function checks to see whether a variable exists and has a a nonempty, nonzero value. It will generates a blank if it returns false.

Reinterpreting Variables

The three functions above will accepts a variable as input and and it will returns the variable’s value converted to the appropriate type.
