PHP与Web开发学习笔记（三）

Posted on 2020-06-29 | In PHP与Web开发学习笔记

字符串操作与正则表达式(二)

substr()函数

用法如下string substr(string string,int start[,int length])

start为起始位置，可用负数。length也可用负数。

字符串比较函数

strcmp()、strcasecmp()、strnatcmp()函数

strcmp(str1,str2)按照字典序排序，如果相同则返回0，若str1在str2后，则返回正数，若相反则返回负数。

strcasecmp()函数将不再区分大小写字母。

strnatcmp()函数将按照人们习惯的方式即自然排序，例如12将大于2.

strlen()函数 用于获取字符串长度

字符串函数匹配和替换字符串

在字符串中查找字符串的函数strstr()、strchr()、strrchr()、stristr()

strstr()函数的原型如：

string strstr(string haystack,string needle[,bool before_needle=false])

haystack 表示要被搜索的字符串，before_needle表示返回那部分的字符串，为true时返回needle之前的部分字符串，为false时返回needle之后的字符串，返回第一个匹配的位置。

stristr()将不分大小。strrchr()将返回最后一个匹配的needle的位置。

正则表达式

正则表达式是一种描述文本所包含模式的方法，描述了一种文本的匹配模式，在PHP中，匹配正则表达式的模式更像是strstr()的匹配模式，而非==的匹配模式。

分隔符

每个正则表达式必须被包含在一对分隔符之中，可以选择非字母、数字、\、和空格的任意符号作为分隔符。一般使用的分割符为/。分隔符后可以添加模式修饰符。

字符类

.可以作为匹配换行符之外的任意字符。

任何在[]中的都将被视为一个字符集，但是将只匹配一个字符

例：

/[aeiou]/将只匹配括号中的元素。可以用-表示一个范围，/[a-zA-Z]/将匹配任意的字母。/[^a-z]/匹配不属于集合中的字符。

部分预定义的集合

类 匹配 类 匹配
[[:alnum:]] 文字数字字符 [[:punct:]] 标点符号
[[:alpha:]] 字母 [[:blank:]] 制表符和空格
[[:asci:]] ASCII字符 [[:lower:]] 小写字母
[[:space:]] 空白字符 [[:upper:]] 大写字母
[[:cntrl:]] 控制符 [[:digit:]] 小数
[[:print:]] 所有可打印的字符 [[:xdigit:]] 十六进制数字
[[:graph:]] 触空格外 [[:word:]] word字符（字母，数字和下划线）
重复

*表示这个模式可以出现0次或更多次 +表示出现一次或更多?表示出现0次或者1次

例：[[:alnum:]]+表示至少一个字母字符

子表达式

()括号中，可以被视为一个表达式与计算中的()相同。例：/(very )* large/可用于匹配"large"、"very large"、"very very large"等

子表达式计数

可以用{}指定子表达式可以出现的次数。例：/(very ){1,3}/表示要出现1-3次，若没有指定上限，则表示至少出现几次，也可以只用一个数字指定出现的次数。

定位到字符串的开头与结尾

^表示该模式的匹配必须出现在字符串的起始位置，$表示该模式出现在字符串的结尾。

例：/^bob/表示匹配开头处为bob的字符串，/com$/匹配以com结尾的字符串，/^[a-z]$/将匹配只有一个字母的字符串。

分支

用|表示模式的选择。例：/com|edu|net/，将匹配这三个字符其中之一。

特殊字符

在匹配特殊字符时，必须在字符前加\作为转义。在PHP中，必须使用单引号来进行模式的引用，如果使用双引号则必须使用双反斜杠进行转义，即使用\\ \\php解释器像将其翻译为\\正则解释器再将其翻译为\。

转义序列

\反斜杠转义符，不仅可以用来进行特殊字符的转义，也就可用与常规字符的转义，例如\n表示换行符\r表示回车符\t表示制表符等，在正则表达式中也出现了一些定义好的字符类型，如下表。

\h 水平空白字符 \V 任意非垂直空白字符
\H 任意非水平空白字符 \w 单词字符
