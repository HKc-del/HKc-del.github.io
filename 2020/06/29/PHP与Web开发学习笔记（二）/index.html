字符串操作与正则表达式(一)

1. 通过htmlspecialchars()过滤输出到浏览器的特殊字符

该函数将把在HTML中具有特殊含义的字符转化成为等价的HTML实体，以防止可能造成的页面布局变化与安全问题。

该函数的函数原型如下

string htmlspecialchars(string string[,int flags = ENT_COMPAT|ENT_HTML401[,string encoding='UFT-8'[,bool double_encode = true]]])

该函数具有一个必选参数以及三个可选参数。

第一个必选参数即string，为目标字符串。第二个参数为可选参数flags，指定了该函数将如何进行翻译，可以使用位掩码来表示可能的组合值。其中flags的可选值如下表所示：

标志 意义
ENT_COMPAT 对双引号编码，不对单引号编码
ENT_NOQUOTES 不对单引号和双引号编码
ENT_QUOTES 对双引号和单引号编码
ENT_HTML401 将代码视作HTML 4.01
ENT_XML1 将代码当作XML 1
ENT_XHTML 将代码视作XHTML
ENT_HTML5 将代码视作HTML 5
ENT_IGNORE 不返回空字符串，并且忽略无效的代码序列
ENT_SUBTITUTE 用unicode替换字符替换无效的代码单元序列
ENT_DISALLOWED 用unicode替换无效的代码点

第三个为可选参数encoding即编码，默认为UTF-8

第四个为可选参数，制定了为是否需要对HTML实体进行编码，默认值为真。

2.其他输出过滤函数

str_replace()函数

其函数原型如下：str_replace(find,replace,string,count)，find为必须参数，为要查找的值，replace为要被替换成为的字符串，string为目标字符串，count为替换数进行计数。

该函数将返回一个被替换完的字符串的拷贝，原字符串将保持不变。

nl2br()函数：他将会把字符串中的\n换行符换成HTML中的<br/>的换行符。

explode()函数

原型如下：

array explode(string separator, string input[,int limit])

其中separator作为分隔符，input作为输入，limit为可选参数，作为块数分割的上限。
