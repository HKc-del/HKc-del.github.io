应用层协议

  1. 公共领域协议

    由RFC文档确定

    可供大家使用

    如：HTTP,SMTP

  2. TCP协议

    面向连接(客户机程序和服务器程序之间必须建立连接)

    可靠的传输服务

    流量控制

    拥塞控制

    没有提供:时延保证、最小带宽

  3. UDP协议

    不可靠的数据传输

    没有提供:建立连接，可靠性，流量控制，拥塞控制，时延和带宽保证

分组交换网络中的延迟

  1. dproc = 节点处理时延－nodal processing delay

    典型几个微秒或更小

  2. dqueue = 排队时延－queuing delay

    取决于路由器的拥塞程度

  3. dtrans = 传输时延－transmission delay

    = L/R, 低带宽链路比较显著

  4. dprop = 传播时延－propagation delay

    几个微妙到数百毫秒（卫星通信高传播延迟）

吞吐量

单位时间内整个网络传输数据的速率或分组数。

  1. 瞬时吞吐量(某一瞬间)

  2. 平均吞吐量(一段时间内)
