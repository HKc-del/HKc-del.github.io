Lecture 2

Public-Key Cryptography

Definition

A public-key encryption scheme is a triple of PPT algorithm (Gen,E,D)

Gen: Gen is the key generation algorithm. Gen(1^k ) outputs a pair (pk, sk)

E: E is the encryption algorithm

D: D is the decryption algorithm. Dsk(C) returns a message m or the symbol ⊥ representing incorrect decryption. Incorrect decryption can happen for example if C is not a valid ciphertext.

Correctness: for all m ∈ M and all possible (pk, sk) output by Gen, we have Dsk(Epk(m)) = m
