Trapdoor Permutations(陷门置换)

Definitions

Trapdoor Permutations family is a tuple of 4 PPT algorithms. !!!!!!!Invertis also a PPT algorithms.

Gen(1^k):Gen is a probabilistic algorithm which outputs a pair (i,td),i is a parameter that indexing a particular permutation fi defined over some domain Di,td is the information that allows the inversion of fi.(Gen是一个生成函数，他将产生一对输出，一个用来对某个群上的某种置换进行索引，一种用于将生成将该种置换逆向所需要的信息即Trapdoor)

Sample(1^k,i):Sample is a probabilistic algorithm that output an element x belongs to Di and x should be uniformly distributed.(Sample是一个均匀分布的随机函数用于在该群上随机产生一个元素x)

Eval(1^k,i,x):Eval is a deterministic algorithm which output a y belongs to Di.Furthermore, for all i output by Gen, the function Eval(1^k,i,.) Di->Di is an permutation. One can view Eval(1^k,i,.) as corresponding to a permutation fimentioned above.

Invert(1^k,i,y):Invert is a deterministic algorithm which output a xbelongs to Di

Correctnesss requirement: Invert(1^k,i,Eval(1^k,i,x))=x

Trapdoor (One-Way) Permutations

The one-way means the fi is hard to invert when the trapdoor information isn’t known.

Formal Definition

The one-way means that any efficient algorithm succeeds in inverting a randomly-generated fi with only very small probability.

efficient means the PPT ones and small means negligible functions
